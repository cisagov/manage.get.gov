Waffle flags control whether certain features are turned on or off in the registrar.

You can adjust settings for specific user groups (superusers, staff), but the setting for "Everyone" takes priority and overrides individual group settings.

If “Everyone” is set to:

  • Unknown > the system will use the specific settings for each user group
  • No > the feature will be turned off for all user groups
  • Yes > the feature will be turned on for all user groups
